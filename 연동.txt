1. Node 서버 생성
    - 필요한 server.js 파일 생성
    - routes, db 등 필요한 모듈을 세팅
    - middle ware 세팅

2. React 프로젝트 생성
    - npx create-react-app 프로젝트이름
    - React 프로젝트는 Single Page Application이지만, 동작하는 부분이 매우 동적
    - npm run build를 통해서 react project를 배포할 수 있는 상태로 만들 것
     ※ build 작업이란 ?
      - 배포를 하기 위한 선행 작업으로, 작성한 코드나 애플리케이션을 모두 모아서 패키징 하는 작업
    - React Project에서 변경사항이 있을 시, 매번 다시 build 해줘야한다.
     ※ 대부분 프로젝트가 마무리 되었을 때 npm run build 작업을 실행함 그렇기 때문에 개발 중에는 보통 서버를 2개 띄워놓고 작업을 하는 편 (nodemon server.js, npm start)
